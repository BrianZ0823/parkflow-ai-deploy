# 招商报告生成器 Skill

## 功能说明

自动从数据库汇总招商数据，生成结构化报告，包含：
- 📊 关键指标概览（入驻率、在谈数、签约数）
- 📈 各阶段企业分布
- 🔄 最近跟进动态
- 🏢 园区资源利用率
- 📝 重点跟进企业列表
- 💡 下一步工作建议

## 使用示例

```
生成一份招商周报
```

```
帮我汇总一下本周的招商数据
```

## 技术实现

- 直接查询 SQLite 数据库获取实时数据
- 自动计算关键招商指标
- 生成 Markdown 格式结构化报告
