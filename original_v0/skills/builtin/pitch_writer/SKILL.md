# 招商话术生成器 Skill

## 功能说明

根据目标企业和园区政策信息，自动生成各类招商话术，包括：
- 招商邮件
- 推介演讲稿
- 招商方案文档

## 使用示例

**用户输入**：
```
给"未来芯片科技"写一封招商邮件，重点介绍我们的免租政策和产业链优势
```

**Skill 输出**：
自动生成包含以下要素的专业邮件：
1. 个性化称呼（根据企业背景）
2. 园区亮点介绍
3. 政策优惠详情
4. 产业链配套优势
5. 联系方式和下一步行动

## 技术实现

- 使用模板引擎（Jinja2）
- 结合企业画像定制内容
- 支持多种话术类型切换
