# 企业匹配分析器 Skill

## 功能说明

对目标企业进行多维度分析，评估其与园区的匹配度：
- 行业匹配度
- 规模匹配度
- 发展阶段匹配度
- 资源需求匹配度

## 使用示例

**用户输入**：
```
分析一下未来芯片科技是否适合入驻我们园区
```

**Skill 输出**：
生成详细的匹配分析报告，包括：
1. 匹配度评分（0-100）
2. 优势分析
3. 潜在风险
4. 入驻建议

## 技术实现

- 基于企业画像和园区资源
- 多维度评分算法
- 生成结构化报告
